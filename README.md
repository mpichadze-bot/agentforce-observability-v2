# Agentforce Observability V2

Multi-Agent Orchestration observability dashboard with AI-powered error explanations.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5180

## Features

- Sessions & Intents management
- Network Graph visualization  
- Analytics Dashboard
- Supervisor View (Hub & Spoke)
- Trace Details with AI explanations
- Intent Analysis

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion
- Vite
